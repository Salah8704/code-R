# Code Route MVP

Starter MVP Next.js pour une appli de révision du code de la route.

## Stack
- Next.js 15
- TypeScript
- Tailwind
- Prisma + PostgreSQL
- NextAuth
- Stripe

## Installation
1. Copier `.env.example` vers `.env`
2. Installer les dépendances:
   npm install
3. Pousser la base:
   npm run db:push
4. Seeder:
   npm run seed
5. Lancer:
   npm run dev

## Déploiement Hostinger
- Créer une base PostgreSQL
- Configurer les variables d'environnement
- Installer les dépendances
- npm run build
- npm run start

Ce projet est un starter MVP.
